Sci-Fi Fonts Collection

A collection of sci-fi themed fonts for your projects

About This Collection

This repository hosts a collection of sci-fi themed fonts that can be used in web projects. The fonts are accessible via GitHub raw URLs, making them easy to embed in your websites or applications.

Available Fonts

Federation

A classic Star Trek inspired font

ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
0123456789

How to Use:

@font-face {
  font-family: 'Federation';
  src: url('https://raw.githubusercontent.com/yourusername/scifi-fonts/main/fonts/federation.ttf') format('truetype');
}

.federation-text {
  font-family: 'Federation', sans-serif;
}

Usage Instructions

Using in HTML/CSS

To use these fonts in your web project:

  1. Add the @font-face declaration in your CSS (see examples above)
  2. Apply the font-family to your elements
  3. Make sure to include fallback fonts for better compatibility

HTML Example:

<h1 class="federation-text">Star Trek Inspired Heading</h1>
<p class="klingon-text">This text will use the Klingon font</p>
<div class="vulcan-text">Logic is the beginning of wisdom, not the end.</div>

Browser Support

These fonts are provided in TTF format, which is supported by most modern browsers. For wider compatibility, you may want to convert them to other formats (like WOFF2, WOFF, EOT) using tools like Font Squirrel's Webfont Generator.
