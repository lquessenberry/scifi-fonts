Usage Instructions

Using in HTML/CSS

To use these fonts in your web project:

  1. Download the font files you need or link directly to the GitHub CDN
  2. Add the @font-face declaration in your CSS (see examples above)
  3. Apply the font-family to your elements
  4. Make sure to include fallback fonts for better compatibility

HTML Example:

<h1 style="font-family: 'TNG Title', sans-serif;">Star Trek: The Next Generation</h1>
<p style="font-family: 'Federation', sans-serif;">United Federation of Planets</p>
<div style="font-family: 'Klingon', sans-serif;">Qapla'</div> <span style="font-family: 'Bajoran', sans-serif;">Deep Space Nine</span>

Using with GitHub Pages

If you're hosting on GitHub Pages, use site-rooted paths:

@font-face {
  font-family: 'TNG Title';
  src: url('/scifi-fonts/webfonts/startrek/title-fonts/TNG_Title/TNG_Title.eot');
  src: url('/scifi-fonts/webfonts/startrek/title-fonts/TNG_Title/TNG_Title.eot?#iefix') format('embedded-opentype'),
      url('/scifi-fonts/webfonts/startrek/title-fonts/TNG_Title/TNG_Title.ttf') format('truetype');
}

Browser Support

These fonts are provided in TTF and EOT formats, which cover most modern browsers. EOT format ensures compatibility with Internet Explorer, while TTF works well in most other browsers.
